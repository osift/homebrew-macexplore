# Osift Macexplore

## How do I install these formulae?

`brew install osift/macexplore/<formula>`

Or `brew tap osift/macexplore` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "osift/macexplore"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
